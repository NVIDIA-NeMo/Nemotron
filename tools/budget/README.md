# custom_logit_processors for nvidia/NVIDIA-Nemotron-3-Nano-30B-A3B-BF16


Install `custom_logit_processors` via pip `pip install -e .` from this folder. This will install the runtime budget control logit processor for V3.

Start a server for a nano v3 checkpoint using the `serve_v3.sh` script.

Once the server is running, you can ping it via the example client script: `client.py`.
