# About Agentic Workflows

_Agentic workflows_ are built from a shared _step_ catalog under `src/nemotron/steps/`.
Each step declares inputs, outputs, and parameters.
_Patterns_ document how to combine steps safely.
Agents and skills, including [**Customize Nemotron**](../customize/index.md), can use the catalog to plan work and generate multi-stage projects.

Use [**Key terms**](key-terms.md) for definitions (*step*, *stage*, *pipeline*, *pattern*, *artifact type*) and to disambiguate recipe stages from catalog steps.
